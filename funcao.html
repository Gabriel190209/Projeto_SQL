Funções agregadas (AVG, COUNT, MAX, MIN, SUM)

 Inicio

As funções agregadas em SQL (como AVG, COUNT, MAX, MIN, SUM) são usadas para realizar cálculos em um conjunto de linhas e retornar um único valor resumido. Elas são ferramentas essenciais para análises e relatórios em bancos de dados, permitindo calcular totais, médias, contagens, valores máximos e mínimos de forma eficiente.

Funções principais:

  • SUM(): Calcula a soma de todos os valores em uma coluna numérica.
  • AVG(): Calcula a média aritmética de todos os valores em uma coluna numérica.
  • COUNT(): Conta o número de linhas em uma tabela ou o número de valores não nulos em uma coluna.
  • MAX(): Retorna o valor máximo em uma coluna.
  • MIN(): Retorna o valor mínimo em uma coluna.

Observações:

  • As funções SUM() e AVG() são aplicáveis apenas a colunas numéricas.
  • MAX(), MIN() e COUNT() podem ser aplicadas a diversos tipos de dados (números, textos, datas, etc.).
  • Valores NULL são geralmente ignorados, exceto pela função COUNT(), que conta apenas os valores não nulos.
  • É possível combinar várias funções agregadas numa consulta:
SELECT COUNT(*) AS num_clientes, AVG(salario) AS media_salario FROM clientes;

Exemplos de código SQL com funções agregadas:

1. COUNT:

SELECT COUNT(*) FROM tabela;
SELECT COUNT(*)FROM tabela WHERE coluna > 10;
SELECT COUNT(DISTINCT coluna)FROM tabela;

2. SUM:

SELECT SUM(coluna)FROM tabela;
SELECT SUM(coluna)FROM tabela WHERE coluna2 = 'valor';

3. AVG:

SELECT AVG(coluna) FROM tabela;
SELECT AVG(coluna) FROM tabela WHERE coluna2 > 5;

4. MAX:

SELECT MAX(coluna) FROM tabela;
SELECT MAX(coluna) FROM tabela WHERE coluna2 < 20;

5. MIN:

SELECT MIN(coluna) FROM tabela;

Exemplo prático:

Suponha que você tenha uma tabela produtos com colunas id, nome, preco e categoria.

SELECT COUNT FROM produtos;
SELECT AVG(*) FROM produtos;
SELECT MAX FROM produtos;
SELECT MIN FROM produtos;
SELECT SUM FROM produtos;
SELECT COUNT FROM produtos WHERE categoria = 'eletronicos';
